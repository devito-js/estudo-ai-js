# Estudo AI JS

Repositorio publico com projetos, experimentos e estudos da minha pos em AI aplicada, usando JavaScript/TypeScript.

## Objetivo

Este monorepo centraliza exercicios praticos e prototipos desenvolvidos ao longo da especializacao.
A ideia e manter em um unico lugar:

- fundamentos de AI aplicada
- integracoes com APIs de LLM
- experimentos com frameworks e bibliotecas
- projetos maiores orientados a jogo e machine learning

## Estrutura

- `fundamentos/`
  - exemplos introdutorios e exercicios base
- `llm/`
  - integracoes e projetos com foco em modelos de linguagem
- `experimentos/`
  - area de testes e prototipos rapidos

## Como executar

Cada subpasta possui seu proprio `package.json`.
Entre no projeto desejado e execute:

```bash
npm install
npm run dev
```

Se nao existir script `dev`, use:

```bash
npm start
```
